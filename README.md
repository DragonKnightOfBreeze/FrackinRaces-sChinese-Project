# 注意

* patch文件结构：op，path，value，origin
* value对应暂未翻译、用于占位的英文文本，或者翻译后的中文文本
* origin对应原始的英文文本
&emsp;
* 如果有对应的参考文档，请参照参考文档进行翻译！！！
* 项目默认换行符为`4个空格长度的换行符`，项目默认行尾为`CRLF`。
&emsp;
* 打包前请全局替换：`"origin"` -> `\\origin`，注释掉英文文本
